# clima-sp

Previsão climática sazonal probabilística para a Região Metropolitana de São Paulo (RMSP).

🌐 **Página pública**: https://agamemnon140.github.io/clima-sp/

Modelo estatístico que estima probabilidades de chuva e temperatura acima/abaixo do normal
em horizontes de 3, 6, 12 e 24 meses, usando índices climáticos globais (ENSO, Atlântico Sul,
Oceano Índico, Oscilação Antártica), com validação honesta da habilidade preditiva em cada horizonte.
